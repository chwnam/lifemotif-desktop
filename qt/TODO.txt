Version 0.1.0021 (Feb. 15th, 2014)
==================================
2014년 1월 26일을 기준으로 1일. 가장 마지막은 이 1월 26일을 기준으로 네번째 자리수 결정.
이것을 적는 날을 기준으로 날짜를 적자. 이전 버전인 0.1.0003은 이전 날짜 언젠가임.
단, 마이너 버전 이상이 바뀌면 리비젼은 그 날짜를 기준으로 0001을 만듦.

< 현재까지 되지 않은 기능들 >
* 목록에서 메시지 아이디 말고, 일련 번호 등으로 나오도록
- 환경설정 윈도우 만들기
- 캘린더 배경색상 변경

< 이번 버전의 문제점 및 해결해야 할 일들>
- 라이센스 조사: GPL, LGPL 라이센스 조항 검사. (위키에 정리)
  - LGPL: qt
  - MIT: mimetic
- 파싱 코드 정리. 현재 간이적으로 처리했으나 최대 기능으로 파싱하도록 API 구성.
  - 인코딩 셋이 올바르지 않아 상당수의 문자열이 깨짐.
  + 이메일의 라벨 또한 디코딩되지 않은 채로 출력됨.
  - 첨부 파일 별도 저장 기능
- 메시지 원본보기 기능
- about 창 제작.
- 텍스트 창의 wordwrap 방식이 요상함.
- 문자 인코딩을 확실하게 설정해야 할 듯.
- 최초 인증받지 못하면 다른 기능은 사용할 수 없어야 하는데, 그렇게 되어 있지 않음.
